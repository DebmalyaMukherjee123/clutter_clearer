# clutter_clearer
Organises your current path files in a specific way based on their extension types.
Solves the clutter issue often happens with our files and folders. It creates folders with the extension names of the files present in your current path directory and arranges your files in those specific folders based on their extension types e.g. .pdf files will be organised in a folder named pdf , .txt ones will be in txt folder and so on.
